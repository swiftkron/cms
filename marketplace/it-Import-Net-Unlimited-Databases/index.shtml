Import IT Professional

IMPORT|IT application gives Act! users the opportunity to have data imported from.CSV, MS Excel, and MS Access files, plus Act!, Oracle and FoxPro databases as well as from ODBC and Navision to Act! fields. All data types can be imported.

In combination with SERVICE|IT add-on, import processes can be scheduled to run automatically and keep your team updated with the latest data, saving their time and your budget.

Being a part of the powerful DATA|SUITE add-on, the IMPORT|IT application lets users smoothly import data from various sources, apply filters to make those data imports more precise, assign data to proper users and link it to the right parent records. Only one license of IMPORT|IT is needed to import data to a master database.

Feature overview for IMPORT|IT

  • Easy definable criteria for data import
  • Duplicate checking through definable parameters
  • Importing of Documents
  • Linking Notes and Histories with parent Contact, Companies or Groups
  • Creating and linking Companies during Contact import
  • Categorization of imported Contacts through static or dynamic grouping
  • Import of data types like Histories, Notes and Activities to Contacts, Groups, Companies or Opportunities with three definable criteria for linking to appropriate parent records
  • Import of all Act! data types including Documents and custom tables
  • Integrated address analysis
  • Custom scripts for selected data import
  • Field content updating
  • Possiblity to skip specific data while running reports
