AutoAdmin for Sage ACT! 2013

The #1 administrative software for Act! is now available for Act!. AutoAdmin III is your "unattended assistant" that automatically backs up and maintains your Act! data, saving you time and preventing disastrous data loss.

  • Easily run administrative tasks automatically with no user intervention on a schedule you control
  • Back up Act! data including contacts, companies, opportunities, notes, histories, document attachments, templates, reports, layouts, and user information
  • Back up multiple workgroup or standalone databases on your network, with options to save each backup to different locations
  • Automatically back up any database to a password-protected offsite FTP site (remote data storage on a secure web server)
  • Back up Act! data even when Act! is open and active, allowing you to continue working without interruption
  • Quickly restore from backups when you have problems with your Act! database
  • Customize the number of backup archives to save to make the most of your hard drive space
  • Automatically “reindex” your Act! database to keep it performing optimally
  • Remove old and unneeded data automatically, such as histories, cleared activities, old opportunities, by custom date ranges
  • Maintain multiple databases with different settings for each database
  • Automatically send activity log files, or just error alerts, after every session to the Act! administrator and other users
  • Schedule and run external applications, such as custom applications or other Act! add-on software programs

System Requirements

  • AutoAdmin III requires Sage ACT! Pro 2013 or Sage ACT! Premium 2013 (installed on same PC, network workstation, or network server)
  • Network access to the location of the workgroup or remote Act! data files, with login passwords
  • Sufficient hard disk space to store your backup files
