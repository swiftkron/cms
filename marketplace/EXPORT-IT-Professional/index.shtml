EXPORT|IT Professional

EXPORT|IT add on allows exporting data from your Act! database to a .CSV file. A rich set of available features lets you not only export the whole dataset, but also apply custom criteria and filter data to make your export more precise to meet your business needs.

Basic data export

  • Contacts, Users, Companies, Groups
  • Exporting by Groups, Secondary Contacts
  • Unique IDs (linking with parent data records)

Communication data export

  • Histories, Notes, Activities
  • Opportunities, Products, Documents

Extended data export

  • Data structures
  • Custom tables and pick-lists

Basic filter criteria and functionality

  • Simple filter
  • Text export with / without formatting
  • Export of related data records

Advanced criteria and functionality

  • Precise definition of exported data sets
  • Selection of fields to export
  • Dates range
  • Automated or manual task execution (with CRMADDON SERVICE|IT)
  • Exported files limit to avoid large data files (Act! 2007 and later)
  • Limiting exported data with help of filters (Act! 2007 and later)
  • Database analysis
  • Act! search export

System Requirements

System requirements are the same as for your Act! system. Supports Act! version starting with Act! 2007.
