TABLE|IT

TABLE|IT allows adding custom tables to Act! on Contact, Company, Group and Opportunity levels. This add-on allows managing tables fast and easily, making the data management process more efficient.

The following features are available with TABLE|IT:

  • Creates and manages new tables with individual fields within the Act! database
  • Creates, changes and deletes user defined tables for Contacts, Groups, Companies or Opportunities
  • Creates, changes and deletes fields
  • Connects Contacts and Documents with specific datasets of the user defined tables
  • Defines the visibility of user defined tables for each user and table
  • Determines the field access and security settings for a single user or a team
  • Creates relations between Contacts, Groups, Companies or Opportunities
  • Creates relations between individual tables - even sub-tables
  • Compiles Contact, Group or Company lookups
  • Adjusts the field design by defining the conditions
  • Exports table views in Microsoft Excel
  • Creates documents and master copies in order to print or transform them in optional formats such as PDF
  • Creates calculations for single fields
  • Search function within the table and filter function of search results
  • Customizes individual list appearance
  • Adjusts dialog view for the management of individual databases
  • Matches search with adjustable criteria within the user defined tables

System Requirements

  • System requirements are the same as for your Act! system. Supports Act! version starting with Act! 2009.
